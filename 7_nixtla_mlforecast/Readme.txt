Nixtla ML FORECAST
Código de para entender cómo utilizar nixtla para forecast de series de tiempo utilizando modelos de machine learning
La librería contiene bastante documentación, este es solo código simple para entender un poco más cómo funciona el package

Documentación original: https://nixtlaverse.nixtla.io/mlforecast/index.html

Explicación notebooks:
- O.x: Los notebooks con el índice 0.x son utilizados para replicar algunos de los notebooks de documentación. Pueden tener algunos comentarios o código extra utilizados para entender de mejor forma cómo funciona la librería

- Resto de notebooks: contiene código propio utilizado para probar el funcionamiento de la librería. En esta carpeta, se probó cómo nixtla utiliza los modelos de ML para hacer forecast de serie de tiempo, si se pueden entrenar de forma independiente, si se pueden entrenar un modelo con múltiple output y tener las diferencias como diferentes features (a fecha actual no se puede), etc
